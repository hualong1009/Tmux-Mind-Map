# 介绍

   我创建了一个仓储，主要是通过思维导图的方式介绍`tmux`工具的一些用法，仓储总包含思维导图的源文件，以及导出的一张PNG图片，用法均是在linux下的`tmux` 1.8做了验证，可以保证文中的命令真实有效。基于仓储中的用法，可以更加高效快捷地使用`tmux`工具。

   仓储中包含一个思维导图源文件，思维导图导出的PNG图片，以及README.md和LISENCE.md
